🎵 Music 명령어

  • !입장: 음성 채널에 들어갑니다.
  • !재생 [URL]: URL을 통해 음악을 재생합니다.
  • !일시정지: 음악을 일시 정지합니다.
  • !재개: 일시 정지된 음악을 다시 재생합니다.
  • !중지: 음악을 멈추고 음성 채널에서 나갑니다.
  • !볼륨 [0-100]: 볼륨을 조절합니다. (기본값: 50)

🛠️ 유틸리티 명령어

  • /출석체크: 하루에 한 번 포인트를 획득합니다.
  • !날씨 [지역]: 특정 지역의 날씨를 확인합니다. (예: !날씨 서울)
  • !청소 [메시지 개수]: 최근 메시지를 삭제합니다. (예: !청소 10)
  • !번역 [문장] [번역 언어]: 문장을 지정된 언어로 번역합니다. (예: !번역 안녕하세요 en)
  • !번역언어: 지원하는 번역 언어 목록을 보여줍니다.
  • !봇상태: 현재 봇의 상태를 표시합니다.
  • /tts설정: 특정 채널을 tts 사용 채널로 지정합니다.
  • /tts해제: tts 사용 채널 지정을 해제합니다.
  • !정보: 봇의 정보를 보여줍니다.

🎮 게임 명령어

  • !주사위: 1에서 6까지의 랜덤 숫자를 출력합니다.
  • !가위바위보 [가위/바위/보]: 가위바위보 게임을 진행합니다. (예: !가위바위보 가위)
  • /도박 [베팅 금액]: 포인트를 베팅합니다. 성공 시 1.5배 (예: /도박 100)
  • /주사위도박 [베팅 금액] [짝수/홀수]: 주사위 게임. 성공 시 1.5배 (예: /주사위도박 100 짝수)
  • /룰렛 [베팅 금액] [0-10]: 룰렛 게임. 성공 시 10배 (예: /룰렛 100 7)
  • /경마 [베팅 금액] [1번-4번말]: 포인트를 배팅하고 경마 게임.
  • /포인트: 자신의 현재 포인트를 확인합니다.
  • /포인트양도: 다른 사용자에게 포인트를 양도합니다. (예: /포인트양도 @유저 100)
  • /랭킹: 포인트 랭킹을 확인합니다. (예: /랭킹 10)
